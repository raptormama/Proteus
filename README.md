# Proteus

## What is it?

**Proteus** is an add-on for Blizzard’s massively multiplayer roleplaying game *World of Warcraft* that is designed to make it easier for players to shrink their character to a small size using the toy [Everlasting Noggenfogger Elixir](https://www.wowhead.com/item=226373).

The elixir, when used, will place one of three random effects on the player character:

- become a skeleton
- gain a slow-fall effect
- shrink in size

It is possible for multiple effects to be applied simultaneously via multiple sips of the potion.

## Who should try this add-on?

Proteus is designed for those who wish to shrink but don’t want either of the other two effects, or for people who don’t want to deal with manually clicking them off.

When activated, Proteus attempts to use the elixir toy. Then it will scan through the player’s active buffs and remove the skeleton or slow-fall effects if they’re found. (If the player is in combat, this removal will take place once combat is complete.)

## How to use it?

Proteus can be activated by clicking an on-screen button or by using a keybind. This keybind is set through the game’s native keybinding interface. The onscreen button can be hidden if desired.

## What does that name mean, anyway?

[Proteus](https://en.wikipedia.org/wiki/Proteus), according to Greek mythology, is a son of the god Poseidon. The name likely derives from the word ρῶτος, or “first”.

*Proteus* is the name of the submarine in the 1966 film *Fantastic Voyage*. In the course of the story, the vessel is miniaturised to a near-microscopic size in order to complete its mission.

**Proteus command line arguments (/proteus ...)**

- toggle: toggle addon functionality
- button: toggle onscreen button
- minimap: toggle minimap icon
- welcome: toggle login welcome message

**Minimap button interactions**

- Left click: toggle visibility of onscreen button
- Right click: toggle addon functionality

**Edit mode**

In edit mode (in the game menu or via `/edit`)

Click on the button in edit mode to:

- scale button
- move button via drag and drop
- toggle addon functionality
- toggle login welcome message

**Masque**

Proteus supports Masque so that it can blend into custom UI designs.
